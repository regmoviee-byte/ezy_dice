# EZY: CYBER DICE

Мини-приложение для Telegram, имитирующее бросок двух кубиков d6 и условный d4. Приложение демонстрирует анимацию броска, учитывает все описанные особые случаи и позволяет добавить пользовательский модификатор к результату.

## Возможности

- Бросок двух d6 с анимацией.
- Автоматический бросок d4 при необходимости (добавляется или вычитается, если выпали 6 или 1).
- Особая синяя хроматическая анимация «Экстремальный успех» при двух 6 и кровавая красная анимация «Экстремальный провал» при двух 1.
- Подсказки для ситуации, когда выпадают одновременно 6 и 1 (d4 не бросается).
- Поле для ввода модификатора, которое сразу пересчитывает итоговую сумму.

## Локальный запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
   ```bash
   git clone <repo-url>
   cd ttrpg_psyho
   ```
2. Откройте `index.html` в браузере или используйте любой статический сервер, например:
   ```bash
   npx serve .
   ```
3. Нажмите кнопку «Бросить кубики», чтобы увидеть анимацию и результаты.

## Подготовка к публикации как Telegram Web App

1. **Задеплойте статический сайт.** Подойдут GitHub Pages, Netlify, Vercel или любой другой хостинг с HTTPS.
2. **Сохраните публичный URL** вашей страницы (например, `https://example.com/ezy-cyber-dice/`).

## Подключение к Telegram

1. **Создайте бота** через [@BotFather](https://t.me/BotFather):
   - Команда `/newbot` → задайте имя и токен.
2. **Создайте Web App-запись** для бота (новая команда BotFather `/newapp`):
   - Введите название `EZY: CYBER DICE`.
   - Вставьте HTTPS-URL, где размещено приложение.
   - Выберите бота, к которому нужно привязать Web App.
3. **Добавьте кнопку запуска Web App:**
   - Через `/setmenubutton` в BotFather установите тип `web_app` и укажите созданное приложение.
   - Либо отправляйте пользователю сообщение с инлайн-кнопкой:
     ```json
     {
       "reply_markup": {
         "inline_keyboard": [
           [
             {
               "text": "EZY: CYBER DICE",
               "web_app": {"url": "https://example.com/ezy-cyber-dice/"}
             }
           ]
         ]
       }
     }
     ```
4. После этого в Telegram появится кнопка, открывающая ваше мини-приложение прямо внутри клиента.

### Советы по кастомизации

- Цвета и эффекты настраиваются в `styles.css`.
- Игровая логика находится в `script.js`, где можно изменить поведение кубов или сообщения.
- Для интеграции с сервером Telegram Web Apps API добавьте JS-скрипт `https://telegram.org/js/telegram-web-app.js` и работайте с объектом `window.Telegram.WebApp`.

Готово! Мини-приложение «EZY: CYBER DICE» можно разворачивать и подключать к вашему Telegram-боту.
