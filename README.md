# EZY: CYBER DICE

Мини-приложение Telegram (Web App) с неоновой анимацией броска двух кубиков d6 и дополнительного d4 по описанным в задании правилам. Приложение полностью написано на нативных HTML, CSS и JavaScript и не требует сборщиков или зависимостей.

## Возможности

- Бросок двух d6 с анимацией и подсветкой результата.
- Автоматическое определение особых случаев:
  - обе шестёрки → хроматический «Экстремальный успех»;
  - обе единицы → «Экстремальный провал» с кровавым свечением;
  - одновременное выпадение 6 и 1 → без d4;
  - отдельные 6 или 1 → автоматический бросок d4 на добавление/вычитание.
- Поле для модификатора, которое суммируется с итогом броска.
- Интерфейс адаптирован под компактный мини-вью Telegram и тёмную тему.

## Локальный запуск

```bash
# в корне репозитория
python3 -m http.server 8080
```

После запуска страница будет доступна по адресу `http://localhost:8080`. Это полезно для тестирования перед публикацией.

## Деплой на GitHub Pages

1. Создайте новый репозиторий на GitHub и загрузите туда файлы `index.html`, `styles.css`, `app.js` и `README.md`.
2. В настройках репозитория включите **Pages** (Settings → Pages → Deploy from a branch) и укажите ветку `main` и корневую директорию `/`.
3. Дождитесь сборки и скопируйте опубликованный URL вида `https://<username>.github.io/<repo>/`.

## Подключение к Telegram как Web App

1. Создайте бота через [@BotFather](https://t.me/BotFather) командой `/newbot` и получите токен.
2. Установите домен Web App командой `/setdomain` → выберите бота → вставьте домен, на котором размещено приложение (например, `https://<username>.github.io`).
3. (Опционально) Установите кнопку в главном меню `/setmenubutton` → выберите `Web App` и вставьте JSON:

   ```json
   {"text": "EZY: CYBER DICE", "web_app": {"url": "https://<username>.github.io/<repo>/"}}
   ```

   Теперь у бота появится кнопка запуска мини-приложения прямо из меню.
4. Чтобы добавить кнопку Web App в конкретный чат, отправьте пользователю сообщение с клавиатурой:

   ```bash
   curl https://api.telegram.org/bot<token>/sendMessage \
     -d chat_id=<chat_id> \
     -d text="Запусти EZY: CYBER DICE" \
     -d reply_markup='{"keyboard": [[{"text": "Запустить", "web_app": {"url": "https://<username>.github.io/<repo>/"}}]], "resize_keyboard": true}'
   ```

   При нажатии кнопки откроется ваше мини-приложение внутри Telegram.

> **Совет:** Telegram кэширует Web Apps, поэтому при обновлении интерфейса добавьте к URL параметр версии (`?v=2`) или измените имя файла, чтобы пользователи получили свежую версию.
